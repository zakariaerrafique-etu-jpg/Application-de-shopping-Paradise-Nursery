# Paradise Nursery

Paradise Nursery is a simple shopping web application built with React.  
It allows users to browse indoor plants, add them to a shopping cart, and manage quantities.

## Features
- Browse plant categories
- Add plants to cart
- Update quantities
- View total price
- Redux state management

## Tech Stack
- React
- Redux Toolkit
- JavaScript
- CSS

## Project Goal
Build a functional shopping cart application for indoor plants with modern UI and state management.
